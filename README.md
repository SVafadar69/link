Link is an open-source home security camera project written in Python, C++, C and SwiftUI (will change to React Native down the road for android users). It employs the SOTA Computer Vision + Digital Signal Processing Algorithms to ensure the maximum amount of safety around the house. 

To begin with, you will need the following parts: 

1. Raspberry Pi 5
2. Hailo AI Accelerator. This project uses 13 TOPS, but you can purchase the 26 TOPS Version, or V.2 for 40 TOPS.
3. Any Camera - you can use CSI or USB.
4. Any 3D print case cut out.
5. Any microphone and speaker array. The Whisplay HAT will be used for this project.
6. Any power supply, and external batteries you see fit. You will need something capable of 5V/A to power the pi continuously and use its AI features.

Should you want a discount on the parts, you are able to purchase the entire bundle here and begin coding -> https://www.vafadarindustries.com/.
